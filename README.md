# EV vs Petrol TCO India

This folder is a standalone static website. The entry file is `index.html`.

## Share it online

Upload the contents of this folder to any static hosting provider such as Netlify, Vercel, GitHub Pages, Azure Static Web Apps, or an internal web server.

The site does not require a backend. All calculations run in the browser.

## Notes

- `Save to default` stores the user's saved assumptions in that browser's local storage.
- `Download PDF` uses the browser print dialog; choose "Save as PDF" to download the dashboard.
- Internet-based car model extraction is represented in the product direction, but this static package does not include a backend web-scraping service.
